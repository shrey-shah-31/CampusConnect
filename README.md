# CampusConnect Placement Portal

## Quick start
1. Import `database/schema.sql` into MySQL.
2. Update credentials in `includes/db.php`.
3. Create an admin user directly in DB (role = admin).
4. Open `http://localhost/CampusConnect/`.

## API endpoints
- `auth/login.php`, `auth/register.php`, `auth/logout.php`
- `api/jobs.php`
- `api/applications.php`
- `api/interviews.php`
- `api/offers.php`
- `api/notifications.php`
- `admin/reports.php`
